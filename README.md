### Provides
- react ^15.x
- react-router ^4.x
- redux ^3.x
- redux-observable ^0.14
- rxjs ^5.x

### Building
- webpack ^2.x

`npm run build`

### Development
- webpack-dev-server ^2.x
- react-hot-loader ^3.0-beta

`npm start`

### Tests
- jest ^20.x
- enzyme ^2.x

`npm test`  
`npm run test:watch`

### Browser Automation
- nightwatch ^0.9
- selenium ^3.4

`npm run test:ui` (with dev-server already running)  
`npm run test:ui:start` (start dev-server, run tests and exit) 

### Requirements
- node `^7.x`
- npm `^4.x`
