### Provides
- react ^15.x
- react-router ^2.x
- redux ^3.x
- redux-observable ^0.12
- rxjs ^5.0

### Building
- webpack ^2.1-beta

`npm run build`

### Development
- webpack-dev-server ^2.1-beta
- react-hot-loader ^3.0-beta
- redux-devtools ^3.x
- reactotron ^1.x (it can be enable in config.js)

`npm start`

### Tests
- jest ^15.x
- enzyme ^2.x

`npm test`  
`npm run test:watch`

### Browser Automation
- nightwatch ^0.9
- selenium ^2.53

`npm run test:ui` (with dev-server already running)  
`npm run test:ui:start` (start dev-server, run tests and exit) 

### Requirements
- node `^6.x`
- npm `^3.x`
